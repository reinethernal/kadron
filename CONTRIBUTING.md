# Вклад в проект

## Создание плагина

1. Скопируйте `plugin_template.py` из корня проекта в каталог `plugins`. Назовите новый файл по шаблону `<your_name>_plugin.py`.
2. Откройте файл и реализуйте методы из шаблона:
   - `register_handlers`
   - `get_commands`
   - при необходимости `get_keyboards`, `on_plugin_load`,
     `on_plugin_unload` и `unregister_handlers`
3. Убедитесь, что файл содержит функцию `load_plugin()`, возвращающую экземпляр вашего класса плагина. Менеджер плагинов использует эту функцию для загрузки.

Файлы, соответствующие шаблону `*_plugin.py`, загружаются автоматически при запуске бота. Если файл удалить или переименовать, плагин будет отключён. Также включить или отключить плагин можно во время работы бота с помощью `PluginManager.unload_plugin()` и `PluginManager.load_plugin()`.

Метод `unregister_handlers(router)` вызывается перед выгрузкой плагина. Он должен удалить из переданного `Router` все обработчики, зарегистрированные этим плагином, чтобы после выгрузки они не вызывались.
